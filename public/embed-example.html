KinetiQ Chat Widget - Embedding Guide

Method 1: Iframe Embed (Recommended)

Embed the chat widget using an iframe:

<iframe 
    src="http://localhost:3012?embedded=true" 
    width="380" 
    height="600" 
    frameborder="0"
    style="position: fixed; bottom: 20px; right: 20px; border-radius: 12px; box-shadow: 0 4px 20px rgba(0,0,0,0.15);"
></iframe>

Method 2: Script Tag Embed

Add this script tag to your HTML page:

<script>
  window.kinetiqChatConfig = {
    apiUrl: 'http://localhost:3001',
    embedded: true
  };
</script>
<script src="http://localhost:3012/kinetiq-chat.js"></script>

Make sure your React app is built and served with the embedded mode enabled.

Method 3: Direct React Component (For React Apps)

If you're using React, you can import the component directly:

import FloatingChatWidget from '@kinetiq/chat-widget';

function App() {
  return (
    <>
      <YourApp />
      <FloatingChatWidget 
        apiUrl="http://localhost:3001"
        sessionId={sessionId}
      />
    </>
  );
}

Configuration Options

You can configure the widget using window.kinetiqChatConfig:

window.kinetiqChatConfig = {
  apiUrl: 'http://localhost:3001',  // Backend API URL
  embedded: true,                     // Enable embedded mode
  position: 'bottom-right',            // Widget position
  theme: 'default'                    // Theme (future feature)
};
