Easy embedding script for any website

üìã Quick Start

Copy and paste this code into your website's <head> section:

<!-- KinetiQ Chat Widget Configuration -->
<script>
  window.kinetiqChatConfig = {
    frontendUrl: 'https://bankaichat.codecafelab.in',  // Your frontend URL (required)
    apiUrl: 'https://bankaichat.codecafelab.in',        // Your backend API URL (update if different)
    position: 'bottom-left',                            // Widget position: 'bottom-right' or 'bottom-left'
    buttonColor: '#4a90e2',                             // Button color (hex code)
    buttonText: 'Chat',                                 // Button text (optional)
  };
</script>
<script src="https://bankaichat.codecafelab.in/kinetiq-chat-embed.js"></script>
üí° Note: The URLs above are already configured with your domain. If your backend API is on a different URL, update the apiUrl value.

Option Type Required Default Description
frontendUrl string Yes - The URL where your frontend is hosted (e.g., https://chat.example.com)
apiUrl string No frontendUrl The backend API URL. If not provided, defaults to frontendUrl
position string No 'bottom-left' Widget position: 'bottom-right' or 'bottom-left'
buttonColor string No '#4a90e2' Button color in hex format (e.g., '#4a90e2', '#ff5722')
buttonText string No 'Chat' Text to display on the button (if you want text instead of icon)
zIndex number No 9999 CSS z-index for the widget (to control layering)

üìù Step-by-Step Instructions

1 Get your URLs

You need two URLs:

  • Frontend URL: Where your chat frontend is hosted (e.g., https://chat.example.com)
  • Backend URL: Your API backend URL (e.g., https://api.example.com)
2 Add the configuration script

Add this to your website's <head> section, before the closing </head> tag:

<script>
  window.kinetiqChatConfig = {
    frontendUrl: 'https://your-frontend-url.com',
    apiUrl: 'https://your-backend-url.com',
  };
</script>
3 Add the embed script

Right after the configuration, add the embed script:

<script src="https://your-frontend-url.com/kinetiq-chat-embed.js"></script>
4 Test it!

Open your website and you should see a chat button in the bottom-left corner. Click it to open the chat widget.

üéÆ JavaScript API

You can control the chat widget programmatically:

// Open the chat widget
window.KinetiQChat.open();

// Close the chat widget
window.KinetiQChat.close();

// Toggle the chat widget
window.KinetiQChat.toggle();

// Get current configuration
console.log(window.KinetiQChat.config);

üêõ Troubleshooting

Widget not appearing?

  • Check browser console for errors
  • Verify the script URL is correct and accessible
  • Make sure the frontendUrl is correct
  • Check if there are any JavaScript errors on your page

Chat not loading?

  • Verify the frontend is running and accessible
  • Check CORS settings on your backend
  • Verify the apiUrl is correct
  • Check browser network tab for failed requests
